## installation
```shell
npm install -S @jswork/react-textarea
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-textarea/dist/style.css";

  // or use sass
  @import "~@jswork/react-textarea/dist/style.scss";

  // customize your styles:
  $react-textarea-options: ()
  ```
2. import js

